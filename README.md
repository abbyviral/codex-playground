# codex-playground

यह प्रोजेक्ट सुमन आचार्य जी के लिए एक सरल ज्योतिष वेब ऐप का डेमो है।

## इस्तेमाल कैसे करें

1. इस रिपॉज़िटरी को क्लोन करें या `.zip` डाउनलोड करें।
2. `index.html` फाइल को किसी भी आधुनिक ब्राउज़र में खोलें।
3. फॉर्म में अपना नाम, जन्मतिथि और जन्म स्थान भरकर "जाने अपनी राशि" पर क्लिक करें।
4. आपको आपकी राशि और कुछ विशेषताएँ हिंदी में दिखाई देंगी।

यह ऐप केवल डेमो उद्देश्यों के लिए है और इसमें कोई सर्वर-साइड कोड नहीं है।
